# MobilityWork Test

## To run the project

1. `make up`
2. `make exec c="composer install"`
3. `make sf c=d:m:m`
4. `make sf c=d:f:l`
5. Go to https://localhost/api/doc to visualize the Nelmio API doc
6. `make unit` to run unit tests
7. Open ./coverage-report/index.html in the browser to visualize test coverage
